# 📚 Lista de Exercícios em Java

Este projeto contém uma série de exercícios desenvolvidos em Java com foco em lógica de programação e cálculos matemáticos básicos.

Todos os exercícios estão organizados em um único arquivo (`App.java`), onde cada método representa um exercício diferente.

---

## 🚀 Objetivo

Praticar conceitos fundamentais da programação, como:

* Entrada de dados com `Scanner`
* Operações matemáticas
* Estruturas sequenciais
* Organização de código em métodos

---

## 📂 Estrutura do Projeto

```java
App.java
```

* O método `main` executa todos os exercícios automaticamente
* Cada exercício está implementado em um método separado (`ex01` até `ex10`)

---

## 🧠 Exercícios Implementados

### 🔹 EX01 - Desconto de 5%

Calcula o desconto de 5% sobre o valor de um produto.

---

### 🔹 EX02 - Conversão Real → Dólar

Converte um valor em reais para dólares com base na cotação informada.

---

### 🔹 EX03 - Combustível

Calcula quantos litros podem ser abastecidos com um determinado valor.

---

### 🔹 EX04 - Pagamento de Pedreiro

Calcula o salário com descontos de:

* INSS (7%)
* IR (15%)

---

### 🔹 EX05 - Acréscimo de 8%

Calcula o valor final de um produto com aumento de 8%.

---

### 🔹 EX06 - Conversão Real → Euro

Converte um valor em reais para euros.

---

### 🔹 EX07 - Conta de Energia

Calcula o valor total com base no consumo de kWh.

---

### 🔹 EX08 - Média Escolar

Calcula a média com pesos:

* P1 = 20%
* P2 = 20%
* P3 = 60%

---

### 🔹 EX09 - Idade em Dias

Converte idade em anos para dias.

---

### 🔹 EX10 - Parcelamento com Juros

Calcula o valor total de um produto parcelado com juros de 2% ao mês.

---

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/yurilirafarias-prog/exercises_java_2.git
```

2. Abra o projeto em uma IDE (ex: IntelliJ ou VS Code)

3. Execute a classe `App.java`

---

## ⚠️ Observação

Todos os exercícios utilizam entrada de dados via terminal.
Ao executar, será necessário inserir os valores conforme solicitado.
